---

## 📌 Project Description

The **Art Tracking System** is a blockchain-based solution that ensures **authenticity, transparency, and secure ownership tracking** of artworks. In traditional art systems, verifying ownership and originality is difficult and prone to fraud.

This project eliminates those issues by using **smart contracts** to permanently store artwork data and transaction history on the blockchain. Every change in ownership is recorded, creating a **trustless and tamper-proof system**.

---

## ⚙️ What It Does

- Registers artwork with unique blockchain IDs  
- Stores artist details and artwork metadata  
- Tracks complete ownership history  
- Verifies authenticity of artworks  
- Enables secure and transparent ownership transfers  

---

## ✨ Features

- 🔐 **Immutable Records**  
  Data stored on blockchain cannot be altered  

- 🎨 **Artwork Registration**  
  Artists can securely register their creations  

- 🔄 **Ownership Transfer**  
  Transparent and verifiable ownership changes  

- 🧾 **Provenance Tracking**  
  Full ownership history of each artwork  

- 🛡️ **Fraud Prevention**  
  Prevents duplication and forgery  

- 🌐 **Decentralized System**  
  No central authority required
